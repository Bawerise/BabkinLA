# Установка WSL2

## Подходит для:
- Windows 10 (версия 2004 и новее, сборка 19041+).
- Windows 11 (WSL2 уже встроен).

---

## Шаг 0. (необязательный, только для Windows 10) Установка Windows Terminal
1. Откройте **Microsoft Store**.
2. В поиске введите **Windows Terminal** и установите приложение терминала.
*(В Windows 11 этот шаг не нужен — приложение терминала поставляется в составе ОС).* 

## Шаг 1. Включение необходимых компонентов Windows  
1. Пуск → Панель управления → Программы → Включение или отключение компонентов Windows 

2. В открывшемся списке отметьте:  
   - **Платформа виртуальной машины (Virtual Machine Platform)**  
   - **Подсистема Windows для Linux (Windows Subsystem for Linux)**  

3. Нажмите **OK** → дождитесь установки компонентов.  
4. Перезагрузите компьютер.  

---

## Шаг 2. Установка обновления ядра WSL2 (только для Windows 10)  
1. Откройте браузер и перейдите на страницу [Linux kernel update package for WSL2](https://aka.ms/wsl2kernel)  
2. Скачайте `пакет обновления ядра Linux WSL2 для компьютеров x64` и установите его через мастер.  
*(В Windows 11 этот шаг не нужен — ядро уже встроено).*  

---

## Шаг 3. Установка WSL через Microsoft Store  
1. Запустите **Microsoft Store**.  
2. В поиске найдите **Windows Subsystem for Linux** и установите приложение.  
   - Это позволит обновлять WSL через Store и управлять им через интерфейс.  

---

## Шаг 4. Установка дистрибутива Linux  
1. В Microsoft Store найдите и установите дистрибутив **Ubuntu 24.04** (рекомендуется)  
2. После установки запустите дистрибутив (через меню Пуск).  
3. При первом запуске система попросит создать пользователя и пароль Linux. Когда вы задаете пароль, он вводится, но не отображается в окнге терминала из соображений приватности. Его необходимо ввести и подтвердить.
