# 6) Очистка

1. Отключите автоочистку
    - в psql проверьте статус автоочистки
    - установите параметр `autovacuum` в значение `off`
    - перечитайте файл конфигурации (с помощью функции `pg_reload_conf`)
    - проверьте статус повторно 
2. Создайте большую таблицу
    - создайте новую базу данных
    - создайте в ней таблицу с единственной колонкой типа `NUMERIC`
    - создайте неуникальный индекс в этой таблице по этой колонке
    - вставьте в таблицу 100000 строк
    Может использоваться конструкция
    ```sql
    INSERT INTO имя_таблицы
    SELECT random() 
    FROM generate_series(1, 100000);
    ```
    - проверьте данные, которыми была заполнена таблица
3. Несколько раз обновите половину строк, каждый раз замеряя размеры таблицы и индекса
    - измерьте исходные размеры таблицы и индекса
    Для замера размера таблицы может использоваться
    ```sql
    pg_table_size(имя_таблицы)
    ```
    Для замера размера индексов может использоваться
    ```sql
    pg_indexes_size(имя_таблицы)
    ```
    - измените половину строк таблицы
    Например, запросом с условием `WHERE имя_колонки < 0.5`
    - замерьте изменения значений размеров
    - повторите 3-4 раза
4. Выполните полную очистку (`VACUUM FULL`) и повторите замер значений размеров
5. Несколько раз обновите половину строк, выполняя после обновления обычную очистку (`VACUUM`), а после нее -- замер размеров
6. Включите автоочистку
    - сбросьте значение параметра `autovacuum`
    - перечитайте файл конфигурации
    - проверьте статус автоочистки (запросом к `pg_settings` с условием `WHERE name='autovacuum'`)
