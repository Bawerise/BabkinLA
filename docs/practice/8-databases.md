# 8) Базы данных и схемы

1. Измерьте размер пустой базы данных
    - с помощью psql создайте новую базу данных
    - подключитесь к ней
    - измерьте ее размер
2. Измерьте размер базы данных с данными
    - создайте в базе данных 2 схемы: `app` и `postgres`
    - создайте в каждой из схем по таблице
    - наполните их данными (например с помощью `generate_series`)
    - измерьте размер базы данных
3. Модифицируйте путь поиска
    - проверьте текущее значение `search_path`
    ```psql
    SHOW search_path;
    ```
    - измените путь поиска таким образом, чтобы таблицы обеих созданных схем были доступны по неквалифицированному пути, при этом приоритет должна иметь схема `postgres`
    - выйдите из psql и войдите повторно
    - проверьте текущее значение `search_path`
    - проверьте наличие доступа по неквалифицированному имени к таблицам в обеих схемах
    - создайте таблицу без указания схемы
    - проверьте, в какую схему она попала
4. Создайте временную таблицу
    - создайте временную таблицу
    - выйдите из psql и войдите повторно
    - проверьте наличие временной таблицы
