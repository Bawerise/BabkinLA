# 11) Низкий уровень

1. Проверьте работу TOAST-таблицы
    - создайте таблицу с колонкой типа `TEXT`
    - проверьте текущую стратегию хранения для колонки
    - измените стратегию хранения на `external`
    - вставьте короткую и длинную (например, с помощью `repeat('guap', 100000)`) строки в таблицу 
    - найдите имя TOAST-таблицы
    ```psql
    SELECT relname
    FROM pg_class
    WHERE oid = (
        SELECT reltoastrelid
        FROM pg_class
        WHERE relname=имя_таблицы)
    ```
    - просмотрите содержимое TOAST-таблицы напрямую с помощью `SELECT` (она находится в схеме `pg_toast`)
2. Сравните размер базы данных и всех таблиц в ней
    - создайте новую базу данных
    - создайте в ней таблицу и наполните ее данными с помощью `generate_series`
    - найдите размер базы данных (с помощью `pg_database_size`)
    - найдите суммарный размер всех таблиц (суммируя `pg_total_relation_size`)
