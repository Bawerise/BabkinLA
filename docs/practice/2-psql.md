# 2) Использование psql

1. Выполните подключение с помощью вспомогательного скрипта (`./sripts/psql.sh`), проверьте информацию с помощью `\conninfo`
2. Выведите список баз данных в подробном виде с помощью `\l+`
3. Настройте файл `.psqlrc` (примечание: он монтируется внутрь контейнера из корня репозитория, поэтому должен создаваться там): 
    - Настройте вид приглашения вида `пользователь@база=#` с помощью переменных pqsl `PROMPT1` и `PROMPT2`
    - Настройте вывод длительности команд в psql с помощью `\timing on`
    - Настройте pager `less -XS` с помощью переменной окружения `PSQL_PAGER`

# Содержимое пояснительной записки
1. Скриншоты с введенными командами, результатами их выполнения и развернутые комментарии к ним
2. Содержимое файла `.psqlrc`
