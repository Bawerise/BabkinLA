# 9) Системный каталог

1. С помощью psql подключитесь к любой базе данных и получите описание таблицы `pg_class`
2. Получите подробное описание представления `pg_tables`
3. Создайте базу данных и временную таблицу в ней. Получите полный список схем в базе, включая системные (командой `\dnS+`)
4. Получите список представлений в схеме `information_schema`
5. Исследуйте список запросов, выполняемых командой psql `\d+ pg_views`
    - включите переменную `ECHO_HIDDEN`
    - выполните команду `\d+ pg_views`
    - выключите переменную `ECHO_HIDDEN`
