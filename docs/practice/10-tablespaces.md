# 10) Табличные пространства

1. Создайте новое табличное пространство и базу данных в нем
    - войдите в консоль сервиса db
    - создайте пустой каталог в каталоге `/tablespaces`
    - в psql создайте новое табличное пространство с расположением в этом каталоге
    - убедитесь, что табличное пространство появилось в системном каталоге
    - создайте базу данных в новом табличном пространстве
    - проверьте содержимое каталога табличного пространства
2. Установите новое табличное пространство в качестве табличного пространства по умолчанию
    - измените для базы данных `template1` табличное пространство по умолчанию на новое табличное пространство
    - создайте новую базу данных без явного указания табличного пространства
    - проверьте табличное пространство по умолчанию у новой базы данных
3. Удалите новое табличное пространство, предварительно удалив базы данных, которые его используют
